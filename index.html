ml by jottinger

Machine Learning in Java

This is a machine learning project, with primary focus on classification engines. The initial classifiers are bayesian in nature, including a naive and fishers classifier, with others to follow.

Dependencies

ML requires a number of libraries, although many of them provide relatively minor dependencies. They include: * Infinispan * Lucene * High-scalability collections

Install

To install ML, you'll need to build from source until it's stable enough to have a maven dependency (which will happen after the perceptron is provided.) The steps are: * Clone the github repository * mvn package
